# optimize-elite-landing-page

## Contexto
Esta tarefa utiliza a expertise do `cro-expert` (Sterling) e `ux-design-expert` (Uma) para transformar uma página de vendas comum em uma máquina de conversão.

## Passos

1. **Auditoria de Hierarquia (Russell Brunson Style)**:
   - Analisar o JSON da página.
   - Verificar se existe: Headline de Impacto, Vídeo/Imagem de Venda, Benefícios vs Características, Garantia e Call to Action (CTA) claro.

2. **Remoção de Atrito (Steve Krug Style)**:
   - Verificar se há links de saída desnecessários.
   - Analisar a velocidade percebida (número de imagens pesadas).
   - Simplificar o formulário de captura/compra.

3. **Injeção de Gatilhos Mentais**:
   - Sugerir locais para depoimentos (Prova Social).
   - Inserir contadores de escassez (se aplicável).

## Output Esperado
Um relatório de modificações priorizadas para a página de vendas.
