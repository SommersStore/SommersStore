# Skill: Componentes UI Luxury

Esta é a skill mestre de VISUAL & EDITORIAL do ecossistema SommersStore Elite.

## 📜 Regras Fundamentais
1. **O que é:** Desenvolvimento de interfaces Frontend com alto refinamento estético, baseadas em Tailwind CSS.
2. **Entradas:** Escopo de Seção, Paleta de Cores do Brandbook (`base`, `primary`, `secondary`, `accent`), referências fotográficas de alta densidade.
3. **Saídas:** Código HTML/React modular, imutável e estruturado para conversão.

## 🗂️ Referenciais Visuais e Contexto
Ao executar esta skill, você *nunca* deve alucinar cores. Você deve obedecer ao `LOCKED_ASSETS.md` para paletas de cores e uso de fontes.
- Fonte Master Intercalada: Somente usar Fontes com Serifas (`Playfair Display` ou similar) para títulos aristocráticos H1/H2 e `Inter` ou `Manrope` para leituralidade de corpo.
- Bordas e Sombras: NUNCA usar bordas de auto-contraste sólidas grossas ou arredondamentos rudes tipo `rounded-3xl`. Use `rounded-sm` ou cantos vivos (zero rounded). As sombras devem ser difusas, profundas: `shadow-2xl` com opacidades sutis (ex: `black/40` ou `white/10`).

## ⚡ Prompt de Execução
> **DIRETRIZ DA SKILL:**
"Escreva CSS/Tailwind como se você estivesse entalhando uma escultura. Para a 'SommersStore', a UI deve ter o efeito '*Glassmorphism* Sutil': fundos profundamente escuros (`bg-[#0a0a0d]`), texturas granuladas e brilhos difusos nos contornos (`border border-white/5`). Qualquer gradiente deve parecer luz batendo lentamente no veludo, evite cores neon berrantes. Não entregue botões largos e arredondados; entregue botões sóbrios, esguios, com tracking expandido (`tracking-widest`), em letras maiúsculas elegantes. O respiro (`padding` massivo) é o maior sinal de riqueza na Web. Se certifique que o componente respira luxo."
