# PremiumHeuristicExtraction

## Purpose
Select only the highest-value material that should be eligible for harmonization.

## Keep
- Decision rules.
- Frameworks and step-by-step methods.
- Distinctive vocabulary.
- Reusable examples.
- Strategic beliefs that change behavior.
- Constraints and anti-patterns.

## Remove
- Filler.
- Generic advice.
- Repeated introductions.
- Unverified claims.
- Style imitation that does not improve decisions.

## Output
Return `approved_payload`, `compressed_payload`, and `excluded_payload` with reasons.
