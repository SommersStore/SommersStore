# RedundancyNoveltyScan

## Purpose
Separate useful novelty from repetition.

## Checks
- New framework not already in clone.
- Stronger phrasing of existing framework.
- Repeated point with no extra value.
- Repeated point with useful example or nuance.
- Contradictory repetition that could confuse the clone.

## Classification
- `NOVO_PREMIUM`
- `NOVO_UTIL`
- `REFORCO_UTIL`
- `REDUNDANTE`
- `CONFLITANTE`

## Output
Return a table of claims with classification and action: keep, compress, archive, or remove.
