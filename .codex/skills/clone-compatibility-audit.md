# CloneCompatibilityAudit

## Purpose
Compare new nectar against the current clone base before any merge.

## Checks
- Voice alignment: tone, posture, vocabulary, level of authority.
- Domain alignment: whether the material belongs to the clone's real operating scope.
- Philosophy alignment: compatible beliefs, mental models, and principles.
- Conflict detection: contradictions against existing canonical instructions.
- Identity contamination: whether another expert voice is leaking into the clone.

## Output
Return compatibility score, compatible payload, conflicts, and any identity-contamination warnings.
