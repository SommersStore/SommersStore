# Image Squad Master Config: Dark Botanical Luxe

Esta configuração define a identidade visual para todas as futuras gerações de imagens do projeto **O Cofre das Botânicas Secretas**.

## 🎨 1. Perfil Estético Geral (The DNA)
- **Estilo**: Fotografia macro cinematográfica, iluminação "Chiaroscuro" (alto contraste entre luz e sombra).
- **Paleta**: Tons profundos de preto basáltico, ouro envelhecido (#C5A059), e toques vibrantes mas contidos da cor do ingrediente (ex: Violeta para Lótus).
- **Atmosfera**: Vapor texturizado, superfícies de madeira escura úmida ou mármore fosco.
