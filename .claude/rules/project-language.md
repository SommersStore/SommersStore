# Project Language Configuration

Primary language: Portuguese (Brazil)

All agents must communicate in Portuguese by default.

Rules:
- All explanations must be written in Portuguese
- All stories must be written in Portuguese
- All documentation must be written in Portuguese
- All discussions must be written in Portuguese

Exceptions (keep in English):
- Code
- File names
- Commands
- Technical identifiers
- Framework names

When interacting with users, always respond in Portuguese unless explicitly requested otherwise.
