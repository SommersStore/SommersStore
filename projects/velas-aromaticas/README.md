# Velas Aromaticas

Produto digital para iniciantes totais que desejam aprender a produzir velas aromaticas, vender as primeiras unidades e construir uma marca artesanal com percepcao premium.

## Direcao

- Publico: iniciante total.
- Promessa: fazer velas, vender velas e montar uma marca.
- Referencia estetica: `O Cofre das Botanicas Secretas` / The Black Protocol.
- Entregavel principal: HTML para area de membros, imprimivel para PDF pelo cliente.
- Frees: checklist de materiais e mini guia de aromas.
- Order bumps: calculadora de preco/lucro e pack de etiquetas/cartoes.
