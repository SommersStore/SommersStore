# SKILL_MASTER_ORCHESTRATOR (O Maestro Atômico V3 Premium)

Este é o protocolo supremo de coordenação. Ele não gera código diretamente, mas orquestra os Agentes e as Skills para criar uma unidade narrativa coesa e de altíssima conversão.

## 🕹️ Ciclo de Trabalho Atômico (Workflow)

### Passo 1: Imersão de Contexto
- O Maestro deve ler o E-book ou PDF fornecido pelo usuário.
- Identificar: Promessa Principal, 3 Dores Fundamentais, Mecanismo Único, Bônus e Oferta.

### Passo 2: Definição Estratégica
- Invocar o **Style Guide Premium** como fonte de verdade.
- **Seleção de Modo:**
    - **MODO ZEN:** Ideal para produtos físicos, bem-estar, natureza (Sommers).
    - **MODO COCKPIT:** Ideal para mentorias, software, técnica e autoridade (AIOX).
- Definir o **Dicionário Visual Atômico** (Ex: "Dark Cockpit + Lime" ou "Zen + Emerald").


### Passo 3: Disparo em Cadeia (The Chain)
Para cada seção, o Maestro invoca o Agente responsável usando o respectivo Protocolo (Skill):

1.  **HERO:** `@copy-strategist (hero-strategist)` + `@art-director` -> `SKILL_HERO_PREMIUM`
2.  **PAIN:** `@copy-strategist (pain-empath)` + `@art-director` -> `SKILL_PAIN_PREMIUM`
3.  **ALCHEMIST:** `@copy-strategist (method-alchemist)` + `@art-director` -> `SKILL_METHOD_PREMIUM`
4.  **VOICES:** `@copy-strategist (pain-empath)` + `@art-director` -> `SKILL_SOCIAL_PROOF_PREMIUM`
5.  **DECISION:** `@copy-strategist (offer-architect)` + `@art-director` -> `SKILL_OFFER_PREMIUM`

### Passo 4: Loop de Refinamento
- O Maestro analisa se a Headline do Passo N conversa com a Sub-headline do Passo N-1.
- Se houver quebra de tom, solicita re-geração ao `@copy-strategist`.

---
## 📜 Regras Inquebráveis
- **Minimalismo de Elite:** Nunca use cores ou sombras que não estejam no Style Guide.
- **Narrativa de Sussurro:** O tom deve ser de autoridade calma, nunca de vendedor agressivo.
- **Aperto Final:** O botão de checkout deve ser o ponto de maior luz e foco da página.
