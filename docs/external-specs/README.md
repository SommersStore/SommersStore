# External Specs (GPT / Grok / Outras IAs)

Coloque aqui os arquivos `.md` gerados por outras IAs (GPT, Grok, etc.) para que o Antigravity leia e implemente.

## Como usar
1. Salve o arquivo `.md` nesta pasta.
2. Diga ao Antigravity: _"Leia o arquivo docs/external-specs/NOME_DO_ARQUIVO.md e implemente."_

## Dica
Peça para a outra IA incluir um **resumo executivo** no início do arquivo para economizar contexto.
