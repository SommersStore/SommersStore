## Descricao

Como operador do AIOX, quero que os botoes externos da subaba `Imp. de Renda` e as linhas de arquivos da declaracao respondam ao clique, para abrir o agente e-CAC, iniciar o preenchimento IRPF2025 ou abrir o arquivo fiscal no Windows.

---

## Criterios de Aceitacao

- [x] `RODAR ECAC AGENT` possui os IDs usados pelo handler e dispara `/api/ecac/run-agent`.
- [x] `PREENCHER IRPF2025` possui os IDs usados pelo handler e dispara `/api/ecac/preencher-irpf`.
- [x] Linhas de arquivos do IR sao botoes clicaveis com caminho do arquivo.
- [x] Backend expoe `/api/ir/open` para abrir arquivos locais permitidos.
- [x] A rota de abertura de arquivo bloqueia caminhos fora das pastas IR/e-CAC permitidas.
- [x] Quality gates (`npm run lint`, `npm run typecheck`, `npm test`) executados.
