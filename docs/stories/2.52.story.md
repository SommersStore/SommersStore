# Story 2.52 - Copia da Celula Anterior em Meses Vazios

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-05-30
**Agente responsavel:** @dev (Codex)

---

## Descricao

Como operador do AIOX, quero que celulas numericas vazias de fevereiro a dezembro copiem o valor da celula anterior, para que a planilha mantenha previsoes mensais sem exigir repeticao manual.

---

## Criterios de Aceitacao

- [x] Celulas numericas vazias de fevereiro a dezembro usam o valor efetivo do mes anterior.
- [x] Janeiro vazio permanece como zero por nao ter mes anterior.
- [x] Valores digitados substituem a copia herdada.
- [x] Holerites anexados substituem a copia herdada nas linhas PM.
- [x] Totais usam o valor herdado para calculo, sem aplicar a regra nas celulas de totalizacao.
- [x] Celulas nao numericas, como labels, ficam fora da regra.

---

## Tarefas

- [x] Diferenciar celula vazia de valor zero.
- [x] Calcular valor efetivo mensal com heranca do mes anterior.
- [x] Permitir limpar uma celula para voltar a herdar o valor anterior.
- [x] Criar novas linhas com fevereiro-dezembro vazios internamente.
- [x] Ajustar PM junho-dezembro para herdar maio ate novo holerite.
- [x] Atualizar testes de integracao.
- [x] Rodar lint, typecheck, teste e validacao visual.
