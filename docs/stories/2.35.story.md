## Descricao

Como operador do AIOX Master, quero que a aba `Construcao` mostre o fluxograma visual do projeto no modelo do `Mapa`, usando a mesma fonte `MAP_DATA`, sem apagar o editor legado.

---

## Criterios de Aceitacao

- [x] `Construcao` renderiza um canvas visual embutido quando o projeto possui `MAP_DATA`.
- [x] O canvas embutido usa nos, arestas, coordenadas, status e labels do Mapa existente.
- [x] O clique em um no do canvas abre detalhes no inspector compacto do Master.
- [x] O editor legado do Mapa continua acessivel por botao `Editor`.
- [x] Projetos sem `MAP_DATA` continuam usando os cards de `project_flows`.
- [x] Validar em navegador local.
