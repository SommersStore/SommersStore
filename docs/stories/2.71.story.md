## Descricao

Como operador da aba Financas, quero que o grupo `Dividas Antigas` fique logo abaixo do bloco `Dividas do Holerite`, permanecendo vazio/manual, para que a organizacao visual mostre primeiro as dividas do holerite, depois o espaco de dividas antigas, e depois as dividas normais.

---

## Criterios de Aceitacao

- [x] `Dividas Antigas` aparece imediatamente apos o bloco `Dividas do Holerite`.
- [x] `Dividas Antigas` continua vazia quando nenhuma linha foi marcada com `oldDebt:true`.
- [x] Dividas normais continuam fora de `Dividas Antigas`.
- [x] Linhas adicionadas ou arrastadas para `Dividas Antigas` aparecem dentro do grupo, antes das dividas normais.
