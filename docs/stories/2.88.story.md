# Story 2.88 - AIOX Trader On Chart v1.20 e trilha MQL5/JForex/ProfitPro

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-06-17
**Agente responsavel:** @dev + Codex

---

## Descricao

Como operador Forex multi-plataforma, quero que o EA AIOX Trader On Chart fique mais legivel no template escuro, abra ordens de forma mais confiavel no MT5 e que o projeto ganhe agentes/skills para MQL5, JForex e ProfitPro, para evoluir o setup MT4 para plataformas mais eficientes.

---

## Contexto

O usuario informou que o EA escuro estava dificil de enxergar sobre o template escuro, que MT5 nao estava abrindo ordens e que MT4 abria ordens com demora. Tambem pediu pesquisa tecnica para enriquecer o conhecimento em MQL5 e iniciar a rota de templates/adaptacoes para JForex e ProfitPro, mantendo o visual MT4 como referencia padrao.

---

## Criterios de Aceitacao

- [x] EA MT4 e MT5 recebem tema claro/bege e escala visual padrao de 115%.
- [x] EA MT5 usa politica de preenchimento por simbolo, preco de mercado padrao no `CTrade.Buy/Sell` e checagem de retcode.
- [x] EA MT4 reduz redesenho pesado no `OnTick` e usa timer para atualizar painel.
- [x] EAs compilam nos terminais Dukascopy MT4/MT5 com 0 errors/0 warnings.
- [x] Existe pesquisa tecnica com fontes oficiais para MQL5, JForex e ProfitPro/NTSL.
- [x] Existem skill/persona novas para MQL5 EA engineering e portabilidade MT4 -> MT5/JForex/ProfitPro.
- [x] Existe template inicial ProfitPro/NTSL para adaptar a Campanha 3 por 1.
- [x] Quality gates principais passam.
