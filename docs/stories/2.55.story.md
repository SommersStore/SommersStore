# Story 2.55 - Formula Direta de Copia Mensal na Planilha

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-06-02
**Agente responsavel:** @dev (Codex)

---

## Descricao

Como operador do AIOX, quero que a coluna de janeiro seja a base da Planilha e que toda celula editavel de fevereiro a dezembro comece como formula de copiar a celula imediatamente anterior, para que a planilha ja abra pronta sem preenchimento manual repetitivo.

---

## Criterios de Aceitacao

- [x] Janeiro fica como base inicial das linhas numericas editaveis.
- [x] Fevereiro a dezembro iniciam como formulas de copia da celula anterior.
- [x] Ao digitar um numero em qualquer mes, esse numero substitui a formula daquele mes.
- [x] Meses seguintes vazios passam a copiar o novo numero digitado.
- [x] Ao limpar uma celula, ela volta a ser formula de copia do mes anterior.
- [x] Linhas de totalizacao de coluna/grupo/fluxo de caixa nao recebem celulas editaveis de formula.
- [x] Holerite PM anexado em maio continua substituindo a formula daquele mes e junho-dezembro copiam maio ate novo holerite.
- [x] Quality gates executados.

---

## Tarefas

- [x] Remover a camada anterior de inferencia de zeros legados.
- [x] Manter a regra simples `null = formula de copiar anterior`, `numero = substituicao digitada`.
- [x] Migrar dados atuais para janeiro como base e fevereiro-dezembro como `null`, preservando holerite PM de maio.
- [x] Validar `Rodoanel 1` copiando janeiro em fevereiro e fevereiro em marco.
- [x] Validar que digitar valor novo em marco faz abril-dezembro copiarem marco.
- [x] Validar totais de linha, grupo e fluxo de caixa.
- [x] Rodar lint, typecheck e teste.
