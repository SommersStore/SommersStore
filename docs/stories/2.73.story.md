# Story 2.73 - Painel Pajero por Blocos Tecnicos

**Status:** Done

**Agente responsavel:** @dev (Codex)

## Descricao

Como operador do projeto Pajero, quero uma aba lateral principal abaixo de Financas com um painel interativo dividido por sistemas tecnicos, para navegar rapidamente entre injecao, correias, cambio, suspensao, sistema de tracao, amortecedores/molas e eletrica.

---

## Criterios de Aceitacao

- [x] A sidebar principal exibe `Pajero` imediatamente abaixo de `Financas`.
- [x] A aba `Pajero` abre o painel interativo do projeto Pajero sem depender do menu legado.
- [x] O painel mostra blocos para `Injecao`, `Correias`, `Cambio`, `Suspensao`, `Tracao`, `Amortecedores/Molas` e `Eletrica`.
- [x] Cada bloco possui estado, prioridade, resumo tecnico, acoes e atalho para artefato/checklist correspondente.
- [x] Selecionar um bloco atualiza o detalhe lateral e preserva o fluxo visual/evidencias existentes do projeto.
- [x] Testes estaticos e validacao visual confirmam a navegacao e renderizacao.

---

## Tarefas

- [x] Promover Pajero para a navegacao lateral principal abaixo de Financas.
- [x] Criar modelo de dados dos blocos tecnicos da Pajero.
- [x] Renderizar grid interativo de blocos e detalhe lateral por sistema.
- [x] Preservar fluxo visual PJ/PO e evidencias reais existentes.
- [x] Atualizar testes estaticos.
- [x] Validar no painel local e rodar quality gates.
