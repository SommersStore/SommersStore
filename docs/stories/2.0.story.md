## Descricao

Como operador do SommersStore, quero que o Master Hub mantenha contexto entre recargas e permita filtros rapidos por item, status e owner, para retomar a analise sem perder tempo.

---

## Criterios de Aceitacao

- [x] Estado de UI do Master Hub persiste em `localStorage` (projeto, modo, item selecionado, filtros).
- [x] Existe filtro por busca textual no canvas.
- [x] Existe filtro por status.
- [x] Existe filtro por owner.
- [x] Existe acao de limpar filtros.
- [x] Caption do modo mostra quantidade filtrada (`x/y itens`).
