# Story 2.9 - Configuracao Manual de Squads com Drag-and-Drop

**Status:** Done  
**Epic:** 2 - MVP do Painel Unico Operacional  
**Criado em:** 2026-04-18  
**Agente responsavel:** @dev (Codex)

---

## Descricao

Como operador do painel, quero configurar squads manualmente com drag-and-drop de agentes e ajustar skills/personas-clones por agente, para montar rapidamente cenarios operacionais sem editar JSON manualmente.

---

## Criterios de Aceitacao

- [x] A aba `Squads` exibe board com colunas por squad e suporte a drag-and-drop de agentes.
- [x] Arrastar um agente entre squads persiste em `docs/control/registry.json`.
- [x] Cada card de agente abre drawer lateral com drag-and-drop para skills (disponiveis <-> vinculadas).
- [x] O drawer lateral permite drag-and-drop de personas (e, por consequencia, clones associados).
- [x] Existe suporte a salvar/aplicar/excluir presets de configuracao de squads.
- [x] Testes de qualidade validam os pontos principais da nova interface.

---

## Tarefas

- [x] Refatorar `renderSquads` para board operacional com colunas de drop.
- [x] Implementar handlers de drag (`dragstart`, `dragover`, `drop`) para mover agente de squad.
- [x] Implementar persistencia de alteracoes do board via `/api/save` em `registry.json`.
- [x] Implementar drawer de configuracao por agente com dual-list drag-and-drop (skills + personas/clones).
- [x] Implementar presets locais de squads (salvar/aplicar/excluir).
- [x] Atualizar checks de qualidade para cobrir os novos ganchos de UI.
