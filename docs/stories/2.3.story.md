## Descricao

Como operador do painel, quero uma timeline por projeto no Master Hub para acompanhar execucoes recentes e estado operacional atual sem sair da visao principal.

---

## Criterios de Aceitacao

- [x] Sidebar do Master Hub exibe secao `Execucoes / Historico`.
- [x] Timeline atualiza ao trocar projeto e modo.
- [x] Timeline tenta usar logs reais associados a IDs do projeto.
- [x] Quando nao houver logs associados, timeline exibe snapshot operacional das etapas do projeto.
- [x] Timeline destaca status por cor (ativo, concluido, bloqueado).
