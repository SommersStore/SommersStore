# Story 2.41 - Fluxo Sequencial Guiado no AIOX Master Next

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-05-24
**Agente responsavel:** @aiox-master + @dev + @ux-design-expert (Codex)

---

## Descricao

Como operador do AIOX Master Next, quero criar um produto em uma esteira sequencial da esquerda para a direita, iniciar pelo primeiro no e avancar automaticamente para o proximo ao concluir cada etapa, para evitar organizacao solta e confusa.

---

## Criterios de Aceitacao

- [x] Painel novo continua isolado em `docs/aiox_master_next.html`.
- [x] Usuario consegue criar um produto guiado com entregavel principal, 2 entregaveis free e 2 order bumps.
- [x] Fluxo criado nasce sequencial, numerado e organizado da esquerda para a direita.
- [x] Primeiro no fica selecionado automaticamente apos a criacao.
- [x] Botao de execucao inicia a etapa quando ela esta pendente/rascunho.
- [x] Ao concluir uma etapa em curso, o proximo no sequencial abre automaticamente.
- [x] Testes de qualidade cobrem o produto guiado e o avanco sequencial.
- [x] Validacao local confirma criacao, start, conclusao, abertura do proximo no e limpeza do estado de teste.
