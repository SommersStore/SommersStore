## Descricao

Como operador do AIOX, quero que a aba Financas seja um painel interativo completo, para inserir dados de renda, gastos, dividas, negociacoes, investimentos e metas futuras com calculos operacionais e persistencia.

---

## Criterios de Aceitacao

- [x] Aba Financas deixa de ser apenas cards estaticos.
- [x] Usuario consegue inserir receitas, gastos, dividas e metas pelo painel.
- [x] Painel calcula KPIs de caixa, divida, desconto, comprometimento e patrimonio projetado.
- [x] Dados persistem em `projects/financas/data/finance_state.json`.
- [x] Painel apresenta trilhas de trabalho para quitacao, investimentos e casa propria.
- [x] Quality gates (`lint`, `typecheck`, `test`) executados.
