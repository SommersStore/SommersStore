## Descricao

Como operador do AIOX, quero anexar o holerite mensal da Policia Militar e ter os creditos e descontos extraidos automaticamente, para que `Pagamento PM` e `Despesas PM` sejam preenchidos sem lancamento manual de cada item.

---

## Criterios de Aceitacao

- [x] O botao global antes de `Planilha` foi removido.
- [x] Cada mes mantem um controle compacto para anexar o holerite sem alargar as colunas.
- [x] O upload do holerite chama um endpoint de extracao automatica.
- [x] Creditos extraidos alimentam `Receitas/Pagamento PM`.
- [x] Descontos extraidos alimentam `Despesas Fixas/Despesas PM`.
- [x] O holerite anexado fica registrado no mes correspondente.
- [x] O holerite de maio/2026 em Downloads foi usado como referencia inicial.
- [x] A planilha mantem os 12 meses dentro da largura horizontal do PC.
