# Story 2.7 - Transcricao Completa de PDF para Personas

**Status:** Done  
**Epic:** 2 - MVP do Painel Unico Operacional  
**Criado em:** 2026-04-18  
**Agente responsavel:** @dev (Codex)

---

## Descricao

Como operador do painel, quero que todo PDF vinculado a uma persona gere automaticamente um Markdown de transcricao completa, para que os agentes tenham contexto profundo e fiel, sem depender apenas de resumos curtos.

---

## Criterios de Aceitacao

- [x] Ao vincular um `book` em PDF, o backend gera um `.md` de transcricao completa.
- [x] O `persona_materials.json` passa a manter trilha dedicada de `full_transcript_files`.
- [x] O drawer de Personas exibe uma nova secao para transcricao completa (PDF -> MD).
- [x] Existe acao explicita para gerar/atualizar transcricoes completas a partir dos livros ja vinculados.
- [x] Workspace de livros referencia o caminho de transcricao completa por livro.
