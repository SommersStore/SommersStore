# Story 2.90 - AIOX Trader On Chart v1.31 layout compacto estilo TOC

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-06-18
**Agente responsavel:** @dev + Codex

---

## Descricao

Como operador Forex com licenca do Trader On Chart, quero que o EA AIOX fique visualmente mais parecido com o painel original e um pouco maior/mais legivel, para usar no MT4/MT5 Dukascopy com menos estranhamento operacional.

---

## Contexto

O usuario enviou screenshots comparando o Trader On Chart original e o AIOX v1.30, pediu para deixar o arquivo o mais parecido possivel e informou que possui licenca. A implementacao manteve codigo proprio, sem decompilar binarios nem copiar marca/assets proprietarios, mas aproximou a disposicao visual do painel: simbolo no topo, versao no canto, linha `Close orders`, campos tabulares, Buy verde, Sell vermelho e rodape discreto.

Tambem foi analisado `Manual-de-Acesso-ao-Sistema-Forex-Facil.pdf`; o PDF tem 1 pagina e trata de acesso/licenca ao conteudo do curso, sem parametros tecnicos de layout do EA.

---

## Criterios de Aceitacao

- [x] EA MT4 e MT5 passam para `#property version "1.31"`.
- [x] Painel usa fundo branco/cinza claro e fonte Arial, aproximando o visual tabular do Trader On Chart.
- [x] Cabecalho exibe simbolo do grafico em azul e versao AIOX no canto direito.
- [x] Linha superior `Close orders:` expõe `All`, `Profit`, `Loss` e `Pend`.
- [x] Modos de risco viram um seletor compacto `Risk % / Risk $ / Lots`.
- [x] SL, TP, trailing, BE, Allow hedge, OCO, modo pendente e distancia em pips ficam em linhas compactas.
- [x] Buy usa verde e Sell usa vermelho, com preco atual dentro dos botoes.
- [x] Fonte e binarios sao instalados/compilados somente nos terminais Dukascopy MT4/MT5.
- [x] Compilacao Dukascopy MT4/MT5 retorna 0 errors e 0 warnings.
