## Descricao

Como operador do painel, quero trabalhar com anexos de persona somente em Markdown/TXT e ter limpeza automatica de imagens/base64, para simplificar a ingestao de conhecimento e reduzir friccao com PDF.

---

## Criterios de Aceitacao

- [x] Drawer de Personas remove opcoes e textos focados em upload/fluxo PDF.
- [x] Menu de anexos de Personas fica orientado a arquivos Markdown/TXT.
- [x] Vinculo de arquivo Markdown executa limpeza automatica (remove imagens/base64).
- [x] Existe acao manual para limpar MDs de uma persona via painel.
- [x] Existe comando CLI para limpeza em lote de MDs de personas.
- [x] Upload legado `book/PDF` bloqueado no backend para evitar bypass do fluxo Markdown-only.
