**Status:** Done

**Agente responsavel:** @dev (Codex)

## Descricao

Como operador da aba Financas, quero que o grupo `Dividas Antigas` comece vazio e receba apenas as dividas que eu escolher, para que as dividas normais permanecam fora dessa subaba ate serem adicionadas ou arrastadas para la.

---

## Criterios de Aceitacao

- [x] `Dividas Antigas` aparece vazia quando nenhuma linha foi marcada explicitamente.
- [x] Dividas normais existentes aparecem fora de `Dividas Antigas`, como estavam antes.
- [x] O grupo `Dividas Antigas` permite adicionar uma nova divida diretamente nele.
- [x] Soltar uma divida normal sobre `Dividas Antigas` move essa divida para o grupo.
- [x] Arrastar uma divida antiga para uma divida normal retira a marcacao de antiga.
- [x] O botao de entrar/sair da somatoria aparece apenas nas linhas marcadas como `Dividas Antigas`.
- [x] Quality gates e validacao no painel local executados.
