**Status:** Done  
**Epic:** 2 - MVP do Painel Unico Operacional  
**Criado em:** 2026-04-19  
**Agente responsavel:** @dev (Codex)

---

## Descricao

Como operador do painel, quero concentrar as abas operacionais secundarias em uma aba principal unica, para reduzir ruido visual na sidebar sem perder funcionalidades.

---

## Criterios de Aceitacao

- [x] Existe uma aba principal unica para o bloco operacional (`Ops Desk`).
- [x] As seis telas (`Alertas`, `Logs`, `Reruns`, `Incidentes`, `Console`, `Why Ran`) ficam como subabas.
- [x] Navegacao por subaba continua abrindo os mesmos panes existentes.
- [x] O estado visual da aba principal acompanha a subaba ativa.
- [x] Quality gates continuam verdes apos a mudanca.
