## Descricao

Como operador do AIOX Master, quero controlar o mapa visual diretamente dentro de `Construcao`, com filtros, salvamento e ajustes protegidos de status, sem depender do pane `Mapa` para operacoes simples.

---

## Criterios de Aceitacao

- [x] `Construcao` exibe filtros nativos de status para o canvas embutido.
- [x] Filtro do mapa permanece no estado do Master via `localStorage`.
- [x] Inspector do no permite trocar status entre concluido, em curso, bloqueado e rascunho.
- [x] Alteracoes feitas no Master persistem no mesmo `/api/map/save` usado pelo Mapa legado.
- [x] Pane `Mapa` antigo continua acessivel como editor completo.
- [x] Validar em navegador local.
