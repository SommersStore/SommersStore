# Story 2.63 - Exclusao Mensal de Holerite PM

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-06-04
**Agente responsavel:** @dev (Codex)

---

## Descricao

Como operador do AIOX, quero excluir o holerite PM somente do mes aberto no modal e diferenciar o botao de `Despesas PM` em verde, para corrigir anexos mensais sem afetar outros meses e melhorar a leitura visual das duas linhas PM.

---

## Criterios de Aceitacao

- [x] O modal de holerite PM tem botao `Excluir holerite deste mes`.
- [x] O botao fica habilitado somente quando o mes possui arquivo, extracao, creditos ou descontos.
- [x] Excluir holerite limpa apenas o mes selecionado: arquivo, creditos, descontos e metadados de extracao.
- [x] Valores de `Pagamento PM` e `Despesas PM` recalculam apos a exclusao mensal.
- [x] Dividas bancarias geradas pelo holerite tem o valor daquele mes limpo, removendo linhas automaticas sem valores restantes.
- [x] O botao mensal de `Despesas PM` fica verde; `Pagamento PM` permanece azul.
- [x] Quality gates e validacao Puppeteer executados.
