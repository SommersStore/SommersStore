# Story 2.66 - Recolher Dividas do Holerite na Planilha

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-06-05
**Agente responsavel:** @dev (Codex)

---

## Descricao

Como operador do AIOX, quero botoes discretos para expandir e retrair as linhas vermelhas vindas do holerite PM em `Despesas PM` e em `Dividas`, para organizar a visualizacao sem alterar os calculos.

---

## Criterios de Aceitacao

- [x] `Despesas PM` tem uma linha compacta com botao discreto para mostrar/ocultar somente os descontos bancarios vermelhos do holerite.
- [x] `Dividas` tem uma linha compacta com botao discreto para mostrar/ocultar somente as linhas copiadas do holerite.
- [x] A seta da linha `Despesas PM` recolhe/expande somente as sublinhas do holerite PM.
- [x] Recolher a linha `Despesas PM` nao oculta Mercado, Celular PJ/PF, Luz + Gas nem as demais linhas de despesas.
- [x] A linha compacta `Dividas do Holerite` continua ocultando somente as linhas vermelhas do holerite quando `Despesas PM` esta aberto.
- [x] Recolher `Dividas` nao oculta as demais dividas.
- [x] Os totais continuam calculados sem depender do estado visual dos botoes.
- [x] Quality gates e validacao Puppeteer executados.
