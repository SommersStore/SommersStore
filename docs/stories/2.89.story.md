# Story 2.89 - AIOX Trader On Chart v1.30 paridade visual e funcional

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-06-18
**Agente responsavel:** @dev + Codex

---

## Descricao

Como operador Forex em MT4/MT5 Dukascopy, quero que o EA AIOX Trader On Chart fique o mais proximo possivel dos recursos publicos do Trader On Chart original, com painel mais legivel e sem funcoes faltando, para operar manualmente no grafico com mais velocidade e confianca.

---

## Contexto

O usuario pediu uma analise dos recursos do arquivo/produto original e que o novo EA nao deixasse faltar funcionalidades visiveis. Como nao foi localizado um arquivo original licenciado claro no workspace, a comparacao foi baseada em pagina publica, screenshots publicos e descricao funcional observavel, sem decompilar binarios nem copiar codigo, marca ou assets proprietarios.

---

## Criterios de Aceitacao

- [x] EA MT4 e MT5 passam para `#property version "1.30"`.
- [x] Painel padrao aumenta para `PanelScalePercent = 130` e preserva fundo claro/bege.
- [x] Painel ganha barra superior com `Market`, `Profit`, `Loss` e `Pending`.
- [x] Painel expõe Risk/Lot, SL Pips, TP Pips, BE, TS, Two Way, OCO, modo de pendente, distancia em pips e botao `Place`.
- [x] Botoes grandes Buy/Sell exibem preco atual no painel.
- [x] MT4 e MT5 suportam seletor de pendente `BUY STOP`, `SELL STOP`, `BUY LIMIT`, `SELL LIMIT`.
- [x] Straddle respeita `Two Way`; desligado, envia apenas o modo pendente selecionado.
- [x] OCO remove pendentes do mesmo simbolo/escopo quando uma posicao e aberta.
- [x] Smart BE considera comissao/swap quando disponivel e aplica buffer extra configuravel.
- [x] Compilacao Dukascopy MT4/MT5 retorna 0 errors e 0 warnings.
