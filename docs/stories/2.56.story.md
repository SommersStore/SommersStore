## Descricao

Como operador do AIOX, quero que os valores das linhas totalizadoras usem a mesma cor do titulo da linha, e que a linha final seja `Fluxo de Caixa`, para melhorar a leitura da Planilha financeira.

---

## Criterios de Aceitacao

- [x] Valores de `TOTAL RECEITAS` ficam na mesma cor azul do titulo.
- [x] Valores de `TOTAL DESPESAS` ficam na mesma cor verde do titulo.
- [x] Valores de `TOTAL DIVIDAS` ficam na mesma cor vermelha do titulo.
- [x] A linha separadora vazia `FLUXO DE CAIXA` foi removida.
- [x] A antiga linha `SALDO DO MES` foi renomeada para `FLUXO DE CAIXA`.
- [x] Valores positivos/zero do fluxo de caixa ficam lilas.
- [x] Valores negativos do fluxo de caixa continuam vermelhos.
- [x] Quality gates executados.
