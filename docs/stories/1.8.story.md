## Descricao

Como operador do SommersStore, quero que os modos `Construcao` e `Vendas` da aba Master sejam abastecidos por dados canonicos por projeto, para atualizar o painel sem hardcode e manter rastreabilidade operacional.

---

## Criterios de Aceitacao

- [x] Existe fonte canonica de fluxos por projeto em `docs/control/project_flows.json`.
- [x] Servidor expone endpoint `GET /api/project-flows`.
- [x] Front da aba Master consome `/api/project-flows` via `loadAllData`.
- [x] Master Hub usa fallback local apenas quando o payload remoto estiver vazio.
- [x] Inspector mostra `dependencias` e `metrica principal` para etapas de fluxo.
