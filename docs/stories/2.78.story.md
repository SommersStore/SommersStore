# Story 2.78 - Botoes Sutis nos Grupos de Dividas

**Status:** Done

**Agente responsavel:** @dev (Codex)

## Descricao

Como operador da aba Financas, quero que as linhas compactas `Dividas do Holerite`, `Dividas Antigas` e `Dividas Atuais` tambem tenham os dois botoes sutis de totalizacao, para gerenciar o grupo inteiro sem precisar alterar linha por linha.

---

## Criterios de Aceitacao

- [x] A linha compacta `Dividas Antigas` mostra os dois botoes sutis de totalizacao.
- [x] A linha compacta `Dividas Atuais` mostra os dois botoes sutis de totalizacao.
- [x] A linha compacta `Dividas do Holerite`, dentro da secao `Dividas`, mostra os dois botoes sutis de totalizacao.
- [x] As tres linhas compactas exibem totais de Jan-Dez e Total anual alinhados com a Planilha.
- [x] Os subtotais das tres linhas compactas usam a cor visual do respectivo grupo.
- [x] Os valores de totais e subtotais ficam alinhados de forma consistente nas colunas.
- [x] Os valores da linha `Fluxo de Caixa` ficam centralizados dentro de cada celula.
- [x] A subaba `Dividas/Acordos` segue a mesma ordem visual de `Planilha > Dividas`.
- [x] O botao de somatoria local remove/restaura todas as linhas do grupo no total de `Dividas`.
- [x] O botao de abatimento em `Receitas` remove o grupo do total de `Dividas` e tambem subtrai seus valores de `Receitas`.
- [x] O estado dos botoes dos grupos e persistido junto com os dados da Financas.
- [x] Quality gates e validacao no painel local executados.
