# Story 2.38 - Modo Edicao Protegido no Mapa Embutido

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-05-24
**Agente responsavel:** @dev + @ux-design-expert (Codex)

---

## Descricao

Como operador do AIOX Master, quero ativar um modo de edicao protegido dentro de `Construcao`, para criar, duplicar e remover nos simples do mapa embutido sem abrir o editor legado para operacoes pequenas.

---

## Criterios de Aceitacao

- [x] `Construcao` exibe um toggle de `Modo edicao` no mapa embutido.
- [x] Estado do modo de edicao persiste no estado do Master.
- [x] `Novo no` so aparece quando o modo de edicao esta ativo.
- [x] Inspector permite duplicar nos quando o modo de edicao esta ativo.
- [x] Inspector permite excluir apenas nos nao-pipeline com confirmacao.
- [x] Nos de pipeline continuam protegidos e devem ser removidos no editor legado.
- [x] Alteracoes continuam persistindo por `/api/map/save`.
- [x] Pane `Mapa` antigo continua acessivel como editor completo.

---
