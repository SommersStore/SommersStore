## Descricao

Como operador do AIOX, quero que os botoes clicaveis da subaba `Imp. de Renda` mantenham a ficha fiscal ativa depois de adicionar ou editar dados, para que o clique pareca imediato e confiavel.

---

## Criterios de Aceitacao

- [x] As abas de fichas fiscais continuam alternando normalmente.
- [x] `+ Adicionar Deducao` adiciona a linha e permanece em `Deducoes`.
- [x] `+ Adicionar Bem ou Direito` adiciona a linha e permanece em `Bens e Direitos`.
- [x] O render do IR preserva a ficha fiscal ativa.
- [x] Quality gates (`npm run lint`, `npm run typecheck`, `npm test`) executados.
