# Story 2.74 - Blocos Pajero em Visao Expandida

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-06-08
**Agente responsavel:** @dev (Codex)

---

## Descricao

Como operador do projeto Pajero, quero abrir cada bloco tecnico em uma visualizacao maior dentro da aba Pajero, para analisar resumo, acoes, evidencias e checklists sem ficar limitado aos cards pequenos.

---

## Criterios de Aceitacao

- [x] Clicar em um bloco tecnico abre uma visao expandida ampla do bloco selecionado.
- [x] A visao expandida mostra titulo, prioridade, estado, resumo, acoes, entrega ligada, evidencias e atalhos principais.
- [x] A visao expandida pode ser fechada sem perder a selecao atual.
- [x] Os cards do grid ficam mais confortaveis para leitura, sem quebrar desktop/mobile.
- [x] Testes estaticos e validacao visual confirmam a abertura expandida.

---

## Tarefas

- [x] Criar overlay/visualizacao expandida para blocos Pajero.
- [x] Ligar clique/teclado dos cards a abertura expandida.
- [x] Ampliar dimensoes e leitura dos cards do grid.
- [x] Atualizar testes estaticos.
- [x] Validar no painel local e rodar quality gates.
