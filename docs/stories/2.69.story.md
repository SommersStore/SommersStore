# Story 2.69 - Grupo Dividas Antigas com Controle de Soma

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-06-07
**Agente responsavel:** @dev (Codex)

---

## Descricao

Como operador da aba Financas, quero uma subaba compacta `Dividas Antigas` dentro da secao `Dividas`, com recolher/expandir e um controle por linha para incluir ou excluir a divida da somatoria da coluna sem apagar seus valores.

---

## Criterios de Aceitacao

- [x] `Dividas Antigas` aparece como grupo compactado separado das `Dividas do Holerite`.
- [x] O grupo `Dividas Antigas` pode ser recolhido e expandido sem afetar as linhas do holerite.
- [x] Cada linha de `Dividas Antigas` tem um botao sutil para entrar/sair da somatoria.
- [x] Ao desligar uma divida antiga, seus valores continuam visiveis, mas deixam de contar em `Dividas` e no `Fluxo de Caixa`.
- [x] Ao religar a divida antiga, seus valores voltam a contar nos totais.
- [x] Quality gates e validacao no painel local executados.

---

## Tarefas

- [x] Auditar a estrutura atual de linhas de `Dividas`.
- [x] Criar grupo `Dividas Antigas`.
- [x] Criar controle por linha para exclusao/inclusao na soma.
- [x] Ajustar totais da secao e fluxo de caixa.
- [x] Atualizar testes estaticos.
- [x] Validar no painel local.
- [x] Rodar lint, typecheck e teste.
