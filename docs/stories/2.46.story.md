# Story 2.46 - Holerite PM Mensal na Planilha de Financas

**Status:** Done
**Epic:** 2 - MVP do Painel Unico Operacional
**Criado em:** 2026-05-30
**Agente responsavel:** @dev (Codex)

---

## Descricao

Como operador do AIOX, quero anexar o holerite mensal da Policia Militar na subaba `Planilha`, para que os creditos alimentem `Receitas/Pagamento PM` e os descontos alimentem `Despesas Fixas/Despesas PM` com rastreabilidade por mes.

---

## Criterios de Aceitacao

- [x] `Planilha` exibe um botao pequeno para acrescentar holerite mensal da PM.
- [x] `Receitas/Pagamento PM` tem seta para expandir/retrair os creditos do holerite.
- [x] `Despesas Fixas` inclui `Despesas PM` como primeira linha.
- [x] `Despesas PM` tem a mesma logica de expansao/retracao de `Pagamento PM`.
- [x] Cada mes permite anexar o holerite correspondente.
- [x] Os creditos e descontos lancados por mes atualizam os totais das linhas-mae.
- [x] Dados de holerite persistem em `projects/financas/data/fin2_data.json`.

---

## Tarefas

- [x] Criar modelo `FIN2_PAYSLIPS` por mes.
