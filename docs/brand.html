Como Falamos

Nossa comunicação é **Séria, Inspiradora e Exclusiva**. Não usamos gírias. Chamamos nossos usuários de "Membros" ou "Alquimistas". Usamos pontuação precisa. Nossa escrita é um convite para o andar de cima.

Nossas Regras

  • 01. Nunca use degradês coloridos. Apenas tons de ouro.
  • 02. O preto deve ser profundo (#050508), nunca cinza.
  • 03. Espaçamento é luxo. Não amontoe informações.
  • 04. Fidelidade total: O que prometemos no Brandbook, entregamos no e-book.
