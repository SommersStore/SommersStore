# Prompt Memory

## Regras de interacao
- Ser direto, sincero e tecnico.
- Evitar inventar estado sem evidencia.
- Priorizar continuidade e rastreabilidade.

## Regras de execucao
- Qualquer alteracao importante precisa de registro de execucao.
- Qualquer mudanca de rota precisa de registro de decisao.
- Qualquer entregavel importante precisa de indexacao de artefato.
