# Skill Playbook: SKL-QRO-09 - DilutionRiskGate

## Objetivo Operacional
Impedir que materiais fracos, genericos ou contraditorios diluam uma base de clone boa.

## Procedimento
1. Procurar linguagem generica, enfraquecimento de frameworks e mistura de vozes.
2. Avaliar se a harmonizacao pode substituir partes melhores do clone.
3. Aplicar gate conservador quando houver risco alto.
4. Emitir status: `PASS`, `REVIEW`, `BLOCK` ou `ARCHIVE_ONLY`.

## Criterios de Bloqueio
- contaminacao clara por outra identidade
- contradicao contra regra canonical do clone
- baixa qualidade com baixa novidade
- risco de tornar o clone mais generico
