# Skill Playbook: SKL-QRO-07 - CloneCompatibilityAudit

## Objetivo Operacional
Comparar o nectar com a base atual do clone para preservar identidade, voz e escopo.

## Procedimento
1. Ler a base atual do clone.
2. Ler o nectar extraido.
3. Verificar voz, dominio, filosofia, contradicoes e contaminacao de identidade.
4. Classificar conflitos como baixo, medio, alto ou critico.

## Saida Esperada
- score de compatibilidade
- trechos compativeis
- conflitos detectados
- alertas de contaminacao de identidade
