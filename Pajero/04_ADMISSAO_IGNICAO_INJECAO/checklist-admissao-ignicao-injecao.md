# Checklist - Admissao, Ignicao e Injecao

| Item | Status | Evidencia | Observacoes |
| --- | --- | --- | --- |
| [ ] Corpo de borboleta | Pendente |  | Sujeira, adaptacao, atuacao |
| [ ] Atuador ou controle de marcha lenta | Pendente |  | Confirmar arquitetura aplicavel |
| [ ] MAF/MAP | Pendente |  | Leitura ao vivo e limpeza apenas se adequado |
| [ ] TPS | Pendente |  | Marcha lenta, progressao, falhas |
| [ ] Entrada falsa de ar | Pendente |  | Mangueiras, juntas, dutos, PCV |
| [ ] Mangueiras de vacuo | Pendente |  | Rachaduras, conexoes, roteamento |
| [ ] Bicos injetores | Pendente |  | Vazao, estanqueidade, padrao de spray |
| [ ] Pressao de combustivel | Pendente |  | Valor exige fonte |
| [ ] Regulador de pressao | Pendente |  | Confirmar arquitetura |
| [ ] Bomba de combustivel | Pendente |  | Pressao, vazao, ruido |
| [ ] Filtro de combustivel | Pendente |  | Local e intervalo exigem fonte |
| [ ] Sondas lambda | Pendente |  | Pre e pos catalisador, resposta e aquecimento |
| [ ] Chicote das sondas | Pendente |  | Emendas, isolamento, conectores |
| [ ] Escapamento sem catalisador | Pendente |  | Registrar fotos e impacto nas leituras |
| [ ] Vazamento antes das sondas | Pendente |  | Pode distorcer mistura |
| [ ] Possivel emulador de sonda | Pendente |  | Procurar emendas/modulos externos |
| [ ] Leitura STFT e LTFT | Pendente |  | Frio/quente, lenta, 2500 rpm e carga |

## Saida esperada

Definir se a marcha lenta e o consumo estao ligados a admissao, ignicao, injecao, escapamento/sondas ou mecanica interna.
