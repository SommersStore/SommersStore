# Checklist - Motor 6G75

## Regra

Sem historico comprovado, tratar itens de sincronismo e manutencao basica como pendentes de validacao.

| Item | Status | Evidencia | Observacoes |
| --- | --- | --- | --- |
| [ ] Oleo do motor | Pendente |  | Nivel, aspecto, cheiro, vazamentos |
| [ ] Filtro de oleo | Pendente |  | Marca/data se houver |
| [ ] Vazamentos | Pendente |  | Tampa, retentores, carter, juntas |
| [ ] Coxins | Pendente |  | Motor e cambio podem afetar trancos |
| [ ] Correia dentada | Pendente |  | Exigir comprovacao ou inspecao |
| [ ] Tensionadores | Pendente |  |  |
| [ ] Polias | Pendente |  |  |
| [ ] Bomba d'agua | Pendente |  | Relacionar com arrefecimento |
| [ ] Velas | Pendente |  | Estado, folga, cor, aplicacao |
| [ ] Bobinas | Pendente |  | Teste e chicote |
| [ ] Cabos, se aplicavel | Pendente |  | Confirmar configuracao |
| [ ] Compressao | Pendente |  | Registrar valores por cilindro |
| [ ] Sincronismo | Pendente |  | Validar se houver sintoma ou intervencao |
| [ ] Sensor de temperatura | Pendente |  | Comparar scanner x real |
| [ ] Sensores de rotacao/fase | Pendente |  | DTCs, dados ao vivo e chicote |
| [ ] Marcha lenta | Pendente |  | Frio/quente, carga, ar-condicionado, direcao |
| [ ] Mistura rica ou pobre | Pendente |  | STFT/LTFT, sondas, MAF/MAP, combustivel |
| [ ] Consumo excessivo | Pendente |  | Correlacionar com DTCs, sondas e escapamento |

## Criterios de validacao

- Marcha lenta estabilizada em frio e quente.
- Ausencia de falhas ativas de ignicao/injecao.
- Temperatura operacional coerente.
- Sem vazamentos criticos apos teste.
