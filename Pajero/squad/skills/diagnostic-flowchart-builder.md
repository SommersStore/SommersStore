# Skill - Diagnostic Flowchart Builder

## Funcao

Criar fluxogramas em Mermaid ou painel visual para ordem de diagnostico.

## Entradas

- sintomas
- dependencias
- gates de seguranca

## Saidas

- fluxograma Mermaid
- passos condicionais
- criterios de avanco/reinspecao

## Donos

- Pajero Chief Diagnostician
- Visual Teardown Mapper

## Bloqueios

Fluxo nao pode pular diagnostico inicial.
