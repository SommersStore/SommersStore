# Skill - Exploded View Matcher

## Funcao

Associar foto real a diagrama ou vista explodida de manual/catalogo.

## Entradas

- foto real
- diagrama
- fonte
- modulo

## Saidas

- correspondencia peca/posicao
- nivel de confianca
- diferencas encontradas

## Donos

- Visual Teardown Mapper
- OEM Technical Librarian

## Bloqueios

Vista explodida sem aplicabilidade V77W nao vira fonte conclusiva.
