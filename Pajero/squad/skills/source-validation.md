# Skill - Source Validation

## Funcao

Classificar informacoes como oficial, tecnica confiavel, secundaria, provavel ou pendente.

## Entradas

- afirmacao tecnica
- fonte
- sistema do veiculo

## Saidas

- status de confianca
- justificativa
- aplicabilidade
- evidencia faltante

## Donos

- Evidence & Anti-Hallucination Auditor
- OEM Technical Librarian

## Bloqueios

Sem fonte, marcar pendente de validacao.
