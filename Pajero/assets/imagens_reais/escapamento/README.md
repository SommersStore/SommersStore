# Fotos Reais - Escapamento

Salve aqui as fotos reais do escapamento da Pajero para substituir os slots pendentes do manifesto.

## Sequencia recomendada

- `01-coletor-esquerdo.jpg`
- `02-coletor-direito.jpg`
- `03-tubos-dianteiros-flanges.jpg`
- `04-sondas-catalisador-ausente.jpg`
