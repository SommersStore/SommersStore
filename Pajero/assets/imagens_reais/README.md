- `cofre-motor-geral-001.jpg`
- `escapamento-sondas-lado-inferior-001.jpg`
- `cambio-v5a51-carter-001.jpg`

## Registro obrigatorio

Toda imagem adicionada deve ser registrada em:

- [Manifesto de imagens](../../data/image-manifest.json)

## Metadados minimos

- modulo
- sistema
- peca
- localizacao
- lado do veiculo, quando aplicavel
- tipo de imagem
- origem
- status de validacao
- observacoes

## Regra

Foto antes de desmontagem sempre que a montagem, roteamento de chicotes, posicao de mangueiras ou orientacao de pecas puder gerar duvida.
