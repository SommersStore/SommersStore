# Checklist - Arrefecimento

| Item | Status | Evidencia | Observacoes |
| --- | --- | --- | --- |
| [ ] Nivel e aspecto do fluido | Pendente |  | Nao assumir especificacao sem fonte |
| [ ] Radiador | Pendente |  | Vazamentos, obstrucoes, colmeia |
| [ ] Tampa do radiador/reservatorio | Pendente |  | Pressao depende de fonte tecnica |
| [ ] Reservatorio | Pendente |  | Trincas, tampa, mangueiras |
| [ ] Mangueiras superiores e inferiores | Pendente |  | Ressecamento, inchaco, abraçadeiras |
| [ ] Valvula termostatica | Pendente |  | Temperatura de abertura exige fonte |
| [ ] Bomba d'agua | Pendente |  | Vazamento, ruido, folga |
| [ ] Ventoinhas | Pendente |  | Acionamento, reles, sensores |
| [ ] Sensor de temperatura | Pendente |  | Comparar scanner x medicao externa |
| [ ] Teste de pressao | Pendente |  | Se equipamento disponivel |
| [ ] Temperatura em rodagem | Pendente |  | Registrar carga, velocidade e clima |

## Alerta

Arrefecimento inseguro bloqueia teste de rodagem longo e qualquer avaliacao agressiva de cambio.
