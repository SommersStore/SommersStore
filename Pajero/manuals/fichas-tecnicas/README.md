# Fichas Tecnicas - Pajero V77W

Esta pasta guarda source cards tecnicos do projeto Pajero. As fichas nao substituem manual oficial, medicao real, scanner ou foto real.

## Regra de uso

- `confirmado`: fonte oficial e aplicabilidade verificada para o escopo.
- `provavel`: fonte tecnica forte, mas ainda exige validacao de aplicabilidade ao veiculo.
- `pendente_de_validacao`: fonte util para orientar busca, mas nao libera compra, torque, fluido, capacidade ou procedimento.

## Escopo fixo

- Pajero HPE 3.8 gasolina 5P 4x4 AT-S.
- Codigo V77W.
- Motor 6G75.
- Cambio V5A51.
- VIN informado: JMYLYV77W5JA00169.
