# Checklist - Freios, Rodas e Rolamentos

| Item | Status | Evidencia | Observacoes |
| --- | --- | --- | --- |
| [ ] Pastilhas dianteiras | Pendente |  | Espessura e desgaste irregular |
| [ ] Pastilhas traseiras | Pendente |  | Espessura e desgaste irregular |
| [ ] Discos dianteiros | Pendente |  | Espessura, empeno, sulcos |
| [ ] Discos traseiros | Pendente |  | Espessura, empeno, sulcos |
| [ ] Pinças | Pendente |  | Travamento e vazamento |
| [ ] Flexiveis | Pendente |  | Rachaduras e vazamentos |
| [ ] Fluido de freio | Pendente |  | Especificacao e troca exigem fonte |
| [ ] Cilindro mestre/servo | Pendente |  | Vazamento, pedal, vacuo |
| [ ] ABS | Pendente |  | DTCs, sensores, chicotes |
| [ ] Rodas | Pendente |  | Trincas, empeno, torque |
| [ ] Rolamentos | Pendente |  | Folga e ruido |
| [ ] Pneus | Pendente |  | Data, desgaste, bolhas, calibragem |
| [ ] Teste de frenagem controlado | Pendente |  | Somente em local seguro |

## Criterio

Freios, pneus e rolamentos precisam estar seguros antes de teste de rodagem mais intenso.
