# Frameworks de Copywriting — Sommers Store

## 1. PAS (Problem → Agitation → Solution)
Usado em: Anúncios, Headlines, Emails iniciais
- **P:** Identifica a dor diretamente
- **A:** Amplifica o custo emocional de não resolver
- **S:** Apresenta o produto como a solução natural

## 2. AIDA (Attention → Interest → Desire → Action)
Usado em: Páginas de vendas longas, VSL
- **A:** Chama atenção com uma headline forte
- **I:** Gera interesse com dados ou história
- **D:** Cria desejo com benefícios e prova social
- **A:** CTA claro e urgente

## 3. Storytelling (Hero's Journey)
Usado em: VSL, Carta de Vendas
- Herói com problema → Encontra o mentor → Recebe o método → Transformação

## 4. Before/After/Bridge
Usado em: Anúncios curtos, Posts
- **Before:** Como é a vida hoje (dor)
- **After:** Como pode ser (desejo)
- **Bridge:** Como o produto conecta os dois

## 5. Features → Benefits → Meaning
Usado em: Bullets de oferta, descrições
- Característica → O que faz → Por que importa para o cliente
