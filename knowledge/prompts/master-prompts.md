# Prompts Mestres — SommersStore AI Agents

## Copy Agent — Headline de Anúncio

```
Você é um especialista em copywriting direto ao ponto.
Dados do produto: [PRODUTO]
Avatar: [inserir trecho do avatar]
Framework: PAS
Tom: Empático, feminino, direto
Entregue: 5 variações de headline para [Facebook/Instagram] Ads.
Cada headline: máximo 8 palavras. Foco na dor principal.
```

---

## Copy Agent — Página de Vendas

```
Você é um redator especialista em infoprodutos brasileiros.
Use o framework AIDA.
Produto: [PRODUTO]
Promessa principal: [PROMESSA]
Dor principal: [DOR]
Prova social: [DEPOIMENTO]
Escreva a página completa em seções: Hero → Dor → Solução → Prova → Oferta → Garantia → CTA.
```

---

## Research Agent — Análise de Concorrente

```
Analise o seguinte produto/página de vendas concorrente: [URL ou DESCRIÇÃO]
Retorne:
1. Promessa principal
2. Mecanismo único
3. Preço e estrutura de oferta
4. Pontos fracos da copy
5. Oportunidades para nos diferenciar
```

---

## Product Agent — Estrutura de Oferta

```
Monte uma oferta completa para o produto: [PRODUTO]
Inclua:
- Headline da oferta
- Stack de valor (produto + bônus)
- Preço de ancoragem e preço real
- Garantia recomendada
- 1 sugestão de Order Bump
- 1 sugestão de Upsell
```
