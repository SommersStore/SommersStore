# Mapa Mental de Conhecimento: DotCom Secrets (Russell Brunson)

Este documento atua como o motor de estrutura de funis para os agentes AIOX.

## 1. A Escada de Valor (The Value Ladder)
- **Bait (Isca)**: O ponto de entrada (Ex: eBook gratuito ou de baixo ticket).
- **Frontend**: O primeiro produto pago (Ex: eBook Sais de Banho).
- **Middle / Backend**: Treinamentos, comunidades, mentorias.
- **Peak**: Produto de alto ticket (Ex: Imersão presencial ou consultoria individual).

## 2. Os 7 Passos de um Funil (The 7 Phases)
1. **Determinar o Tráfego**: Quente, Morno ou Frio.
2. **Ponte de Entrada**: Pré-frame da oferta.
3. **Qualificar Inscritos**: Landing Page / Squeeze Page.
4. **Qualificar Compradores**: Tripwire / Oferta de Entrada.
5. **Identificar Compradores Hiperativos**: Upsells e Order Bumps.
6. **Age & Ascend**: Mover o cliente na Escada de Valor.
7. **Mudar o Ambiente de Venda**: E-mail para Vídeo, Vídeo para Chamada.

## 3. Estruturas de Funil Comuns
- **Funil de Tripwire**: Venda rápida com Upsells.
- **Funil de Webinar**: Venda de tickets médios/altos via educação.
- **Funil de Lançamento (PLF)**: Construção de antecipação massiva.
